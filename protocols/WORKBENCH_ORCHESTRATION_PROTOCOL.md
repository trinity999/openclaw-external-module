# Workbench Orchestration Protocol

- Work orders are created in `work_orders/WO-XXXXX.md`.
- Artifacts are expected in `artifacts/WO-XXXXX/`.
- Completion condition: `artifacts/WO-XXXXX/report.md` exists.
- Ingestion writes registry updates in `registry/work_order_registry.json`.
- Do not modify completed artifacts.
